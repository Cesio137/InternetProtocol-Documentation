## How to use

<Steps>
  <Step>
    Blueprint: search by [`GetInternetProtocolSubsystem`](/docs/Unreal/subsystem#how-to-use) and call [`CreateTCPClientSsl`](/docs/Unreal/subsystem#create-tcp-client-ssl) function
  </Step>
  <Step>
    C++: Use the code bellow
    ``` cpp
    #include "TCP/TCPClient.h"

    UTCPClientSsl* TcpClientSsl = NewObject<UTCPClientSsl>();
    ```

  </Step>
</Steps>
