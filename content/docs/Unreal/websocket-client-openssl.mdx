## How to use

<Steps>
  <Step>
    Blueprint: search by [`GetInternetProtocolSubsystem`](/docs/Unreal/subsystem#how-to-use) and call [`CreateWebsocketClientSsl`](/docs/Unreal/subsystem#create-websocket-client-ssl) function
  </Step>
  <Step>
    C++: Use the code bellow
    ``` cpp
    #include "Websockets/WebsocketClient.h"

    UWebsocketClientSsl* WebsocketClientSsl = NewObject<UWebsocketClientSsl>();
    ```
  </Step>
</Steps>

## Functions

This class has the same functions from ```WebsocketClient```, except for the ones shown below

### Load Private Key Data

* Load the private key from a string.
* If any error occurs, it will trigger the ```OnError``` event.

<ImageZoom src="/img/client-side/websocket/ssl/loadkeydata.png" width={230} height={134} />

### Load Private Key File

* Load the private key from file.

* Relative path.
