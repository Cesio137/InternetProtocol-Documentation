## How to use

<Steps>
  <Step>
    Blueprint: search by [`GetInternetProtocolSubsystem`](/docs/Unreal/subsystem#how-to-use) and call [`CreateHttpServer`](/docs/Unreal/subsystem#create-http-server) function
  </Step>
  <Step>
    C++: Use the code bellow
    ``` cpp
    #include "http/httpserver.h"

    UHttpServer* net = NewObject<UHttpServer>();
    ```
  </Step>
</Steps>

### Backlog

-    Set/Get the maximum number of simultaneous client connections the server will accept in queue.

### Open

-   Opens the TCP server and starts listening for incoming connections.
-   It returns false if socket is already open or if asio return any error code during the bind.

<UFunction
  name="Open"
  className="Http Server"
  pure={false}
  inPins={[
    { name: "", pintype: "exec" },
    { name: "Target", pintype: "object" },
    { name: "Bind Opts", pintype: "struct" }
  ]}
  outPins={[
    { name: "", pintype: "exec" },
    { name: "Return Value", pintype: "bool" }
  ]}
/>

### Close
