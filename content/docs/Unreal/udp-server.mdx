## How to use


### Send To

-   Broadcasts a string to an specific endpoint.
-   It returns false if socket is closed or if buffer is empty.

<UFunction
  name="Send To"
  className="UDPServer"
  pure={false}
  inPins={[
    { name: "", pintype: "exec" },
    { name: "Target", pintype: "object" },
    { name: "Message", pintype: "fstring" },
    { name: "Callback", pintype: "delegate" }
  ]}
  outPins={[
    { name: "", pintype: "exec" },
    { name: "Return Value", pintype: "bool" }
  ]}
/>

  outPins={[

### Bind

-   Listen for datagram messages on a named port and optional address.
-   It returns false if socket is already open or if asio return any error code during the bind.

<UFunction
    name="Bind"
    className="UDPServer"
    pure={false}
    inPins={[
        { name: "", pintype: "exec" },
        { name: "Target", pintype: "object" },
        { name: "Bind Opts", pintype: "struct" },
    ]}
    outPins={[
        { name: "", pintype: "exec" },
        { name: "Return Value", pintype: "bool" },
    ]}
/>

### Close

-   Close the underlying socket and stop listening for data on it. 'on_close' event will be triggered.

<UFunction
    name="Close"
    className="UDPServer"
    pure={false}
    inPins={[
        { name: "", pintype: "exec" },
        { name: "Target", pintype: "object" },
    ]}
    outPins={[{ name: "", pintype: "exec" }]}
/>

## Events

### On Listening

-   This event will be triggered when socket start to listening.

<UEvents
    name="On Connected"
    showBindNode={true}
    outPins={[{ name: "", pintype: "exec" }]}
/>

### On Message

-   Event triggered when socket send or receive data.

<UEvents
    name="On Message"
    showBindNode={true}
    outPins={[
        { name: "", pintype: "exec" },
        { name: "Buffer", pintype: "bytearray" },
        { name: "Bytes Recvd", pintype: "int" },
        { name: "Endpoint", pintype: "struct" }
    ]}
/>

### On Close

-   Event triggered when socket is closed.

<UEvents
    name="On Close"
    showBindNode={true}
    outPins={[{ name: "", pintype: "exec" }]}
/>
