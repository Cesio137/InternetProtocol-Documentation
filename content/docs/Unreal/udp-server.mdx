## How to use

<Steps>
  <Step>
    Blueprint: search by [`GetInternetProtocolSubsystem`](/docs/Unreal/subsystem#how-to-use) and call [`CreateUDPServer`](/docs/Unreal/subsystem#create-udp-server) function
  </Step>
  <Step>
    C++: Use the code bellow
    ``` cpp
    #include "udp/udpserver.h"

    UUDPServer* net = NewObject<UUDPServer>();
    ```

  </Step>
</Steps>
