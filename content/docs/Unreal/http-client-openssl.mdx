## How to use

<Steps>
  <Step>
    Blueprint: search by [`GetInternetProtocolSubsystem`](/docs/Unreal/subsystem#how-to-use) and call [`CreateHttpClientSsl`](/docs/Unreal/subsystem#create-http-client-ssl) function
  </Step>
  <Step>
    C++: Use the code bellow
    ``` cpp
    #include "Http/HttpClient.h"

    UHttpClientSsl* HttpClientSsl = NewObject<UHttpClientSsl>();
    ```
  </Step>
</Steps>
