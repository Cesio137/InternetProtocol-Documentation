## How to use

<Steps>
  <Step>
    Blueprint: search by [`GetInternetProtocolSubsystem`](/docs/Unreal/subsystem#how-to-use) and call [`CreateWebsocketServerSsl`](/docs/Unreal/subsystem#create-websocket-server-ssl) function
  </Step>
  <Step>
    C++: Use the code bellow
    ``` cpp
    #include "WS/WSServer.h"

    UWSServerSsl* WSServerSsl = NewObject<UWSServerSsl>();
    ```

  </Step>
</Steps>
