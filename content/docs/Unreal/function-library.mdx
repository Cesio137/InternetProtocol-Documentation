## Nodes

### Join Threads

- Block execution until all asynchronous tasks on the sockets are completed.
- Make sure the socket's Close function is called beforehand.

<UFunction
  name="Join Threads"
  pure={false}
  inPins={[

  ]}
  outPins={[
    { name: "", pintype: "exec" }
  ]}
/>

- Stop all async operation at once.
