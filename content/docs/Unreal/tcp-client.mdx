## How to use

<Steps>
  <Step>
    Blueprint: search by [`GetInternetProtocolSubsystem`](/docs/Unreal/subsystem#how-to-use) and call [`CreateTCPClient`](/docs/Unreal/subsystem#create-tcp-client) function
  </Step>
  <Step>
    C++: Use the code bellow
    ``` cpp
    #include "TCP/TCPClient.h"

    UTCPClient* TcpClient = NewObject<UTCPClient>();
    ```
  </Step>
</Steps>
