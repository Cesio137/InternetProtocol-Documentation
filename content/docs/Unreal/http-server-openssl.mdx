## How to use

<Steps>
  <Step>
    Blueprint: search by [`GetInternetProtocolSubsystem`](/docs/Unreal/subsystem#how-to-use) and call [`CreateHttpServerSsl`](/docs/Unreal/subsystem#create-http-server-ssl) function
  </Step>
  <Step>
    C++: Use the code bellow
    ``` cpp
    #include "Http/HttpServer.h"

    UHttpServerSsl* HttpServerSsl = NewObject<UHttpServerSsl>();
    ```

  </Step>
</Steps>
