## How to use

<Steps>
  <Step>
    Blueprint: search by [`GetInternetProtocolSubsystem`](/docs/Unreal/subsystem#how-to-use) and call [`CreateJsonObject`](/docs/Unreal/subsystem#create-json-object) function
  </Step>
  <Step>
    C++: Use the code bellow
    ``` cpp
    #include "json/json.h"

    UJavaScriptObjectNotation* Json = NewObject<UJavaScriptObjectNotation>();
    ```

  </Step>
</Steps>
