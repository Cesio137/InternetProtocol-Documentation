## C++ Requirements

> You must follow these requirements before you begin
>  - Your project must use C++ 17 or above
>  - Requires [InternetProtocol](https://github.com/Cesio137/InternetProtocol/releases)
>  - Requires ([ASIO non-boost](https://sourceforge.net/projects/asio/files/asio/) 1.30.x or above)
>  - ([Openssl](https://www.openssl.org/) is only required if you need to use it within the library.)

## How to use

<Steps>
  <Step>
    Download [`InternetProtocol`](https://github.com/Cesio137/InternetProtocol/releases) and [`ASIO non-boost`](https://sourceforge.net/projects/asio/files/asio/)
  </Step>
  <Step>
    Extract both headers to include folder of your project
  </Step>
  <Step>
    You need to build [`Openssl`](https://www.openssl.org/) and link its libraries to your project yourself if you wish to use it.
  </Step>
</Steps>

## Unreal Engine Requirements

> You must follow these requirements before you begin
>  - Your project must use unreal engine 4.27 or above
>  - Requires [InternetProtocol Plugin](https://github.com/Cesio137/InternetProtocol/releases) or from ([Fab marketplace](https://github.com/Cesio137/InternetProtocol/releases)) if you wish to support me
